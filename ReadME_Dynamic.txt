Distribuição Ótima de Sensores

Este é um programa Python que otimiza a distribuição de sensores para monitorar uma série de pontos no plano. Ele oferece a opção de inserir manualmente as coordenadas dos pontos ou gerar aleatoriamente os pontos dentro de um limite superior definido pelo usuário. Em seguida, calcula a distribuição ótima de sensores usando programação dinâmica e exibe as distâncias entre todos os pares de pontos, juntamente com a distribuição dos sensores.

Funcionalidades

Limpeza do terminal para manter a interface do usuário limpa.
Cálculo da distância euclidiana entre dois pontos.
Geração de pontos aleatórios dentro de um limite superior definido.
Solicitação da entrada manual das coordenadas dos pontos.
Otimização da distribuição dos sensores usando programação dinâmica.
Menu interativo para permitir ao usuário escolher entre entrada manual e geração aleatória de pontos.
Exibição das distâncias entre todos os pares de pontos.
Exibição da distribuição ótima dos sensores.
Pré-requisitos

Python 3.x
Biblioteca NumPy
Como Usar

Clone o repositório ou faça o download do arquivo do script.
Certifique-se de ter o Python instalado em seu sistema.
Instale a biblioteca NumPy, se ainda não estiver instalada:
Copiar código
pip install numpy
Execute o script Python:
Copiar código
python sensor_distribution.py
Siga as instruções apresentadas no menu para inserir manualmente as coordenadas dos pontos ou gerar aleatoriamente os pontos.
Insira o número de sensores disponíveis quando solicitado.
Visualize as distâncias entre os pontos e a distribuição ótima dos sensores.
